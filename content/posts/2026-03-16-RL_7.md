# DDPG与TD3：连续动作空间的深度确定性策略梯度

本文介绍DDPG（Deep Deterministic Policy Gradient）和TD3（Twin Delayed DDPG）算法，它们是处理连续动作空间的经典深度强化学习算法。

## 1. 背景与动机

在前面的[文章](https://warden2018.github.io/2025/06/27/2025-06-27-RL_4/)中，我们介绍了DQN，它们在离散动作空间表现出色。然而，在许多实际控制任务中（如机器人控制、自动驾驶等），动作空间是连续的。DQN无法计算所有的$Q(s,a)$,代价太大。随着动作空间维度的提升，离散化动作空间带来的成本增加呈指数级别。

DDPG正是为解决连续动作空间控制问题而设计的。

## 2. DDPG核心思想
参考[Policy Gradient](https://warden2018.github.io/2025/06/09/2025-06-09-RL_5/)和[PPO](https://warden2018.github.io/2026/01/15/2026-01-15-RL_6/)，DDPG和TD3想要通过梯度下降的方法对Policy做优化(调节$\theta$)，在训练结束的时候，具有一个最优化的$\theta$，使得$Q(s,a)$$\big(a=\mu_\theta(s)\big )$最大化，还要借鉴DQN当中使用到的replay buffer: $\mathcal D$和target network等技术，来提升训练的稳定性和效率。
所以优化的目标就是: $Q^\star(s,a)\approx Q^\star(s,\mu_\theta(s))$

- 主要目标是优化一个actor网络$\mu_\theta(s)$，使得在状态$s$下输出的动作$a$能够最大化$Q$值。
- 优化过程，为了能够体现什么是好的$Q$值和当前策略下的$Q$值，需要一个网络$Q_\phi(s,a)$作为评价当前policy的critic，还需要一个能够体现$Q^\star(s,a)$的网络(target_critic)：$Q_{\phi_{targ}}(s,a)$。类似于DQN,我们可以借助MSBE(mean-squared Bellman Error)

$$
\mathcal L(\phi,\mathcal D) = \mathbb E_{(s,a,r,s')\sim \mathcal D} \bigg[ \bigg( r + \gamma(1-d) Q_{\phi_{targ}}(s',\mu_{\theta_{targ}}(s')) - Q_\phi(s,a) \bigg)^2 \bigg]
$$

$d=1 \ if \ s' \ is \ terminal \ else \ 0$，$\mathcal D$是经验回放缓冲区。

$Q_{\phi}(s,a)$更新之后，优化Actor网络$\mu_\theta$的目标是：

$$
\mathcal L(\theta,\mathcal D) = \mathbb E_{(s,a,r,s')\sim \mathcal D} \bigg[ \nabla_\theta \mu_\theta(s) \bigg]^\top \nabla_a Q_{\phi_{targ}}(s,a) \bigg|_{a=\mu_\theta(s)}
$$




### 2.1 确定性策略

与PPO等随机策略不同，DDPG采用**确定性策略**（Deterministic Policy）：

$$
a = \mu_\theta(s)
$$

这意味着给定状态时，Actor直接输出一个确定的动作，而不是动作的概率分布。

### 2.2 Actor-Critic架构

DDPG采用经典的Actor-Critic架构：

- **Actor（策略网络）**：$\mu_\theta(s)$ - 输入状态，输出确定性动作
- **Critic（价值网络）**：$Q_\omega(s, a)$ - 输入状态和动作，输出Q值

Critic网络的输入是**状态+动作**的组合，这是与PPO的重要区别。

### 2.3 策略梯度

DDPG的策略梯度公式为：

$$
\nabla_\theta J(\theta) \approx \mathbb{E}_{s \sim \rho^\mu}[\nabla_a Q_\omega(s, a)|_{a=\mu_\theta(s)} \nabla_\theta \mu_\theta(s)]
$$

直观理解为：首先用Critic评估在当前状态$s$下采取动作$a$的价值，然后沿着使Q值最大的方向更新策略参数。

### 2.4 经验回放与目标网络

DDPG借鉴了DQN的两个关键技术：

1. **经验回放（Replay Buffer）**：存储$(s_t, a_t, r_t, s_{t+1})$ tuples，训练时随机采样，打破时间相关性
2. **目标网络（Target Network）**：使用延迟更新的目标网络计算TD目标，提高训练稳定性

#### 经验回放的四大优势

使用replay buffer有以下几个关键优势：

1. **打破时序关联**：一条轨迹中的$(s_t,a,r,s_{t+1})$具有强相关性，直接用于训练会导致网络难以收敛。随机采样打散了这种关联。

2. **提高数据效率**：在奖励稀疏的环境中（如机器人抓取），智能体可能需要探索很长时间才能获得少量有价值的transition。经验回放可以重复利用历史数据，多次从中采样进行更新，显著降低了数据采集成本。

3. **支持优先采样**：可以在不同训练阶段选择更有价值的transition进行学习。例如，当需要更多更新Actor时，可以优先选择那些更能提升策略的样本。

4. **容纳多样化经验**：回放缓冲区可以存储来自不同策略产生的数据，使最终学到的策略能够适应多变的环境，增强泛化能力。

### 2.5 探索策略

由于采用确定性策略，DDPG需要额外的探索机制：

$$
a = \mu_\theta(s) + \mathcal{N}
$$

在动作上添加噪声（通常为Ornstein-Uhlenbeck噪声），促进探索。

## 4. TD3：双延迟深度确定性策略梯度

TD3是DDPG的改进版本，针对DDPG的Q值过估计问题提出了三项关键改进。

### 4.1 三大改进

#### 1. 截断双Q学习（Clipped Double Q-Learning）

使用两个独立的Critic网络，取较小的Q值作为目标：

$$
y = r + \gamma \min_{i=1,2} Q_{\omega_i'}(s', \mu_{\theta'}(s') + \epsilon)
$$

其中$\epsilon \sim \text{clip}(\mathcal{N}(0, \sigma), -c, c)$为目标策略平滑正则化。

#### 2. 延迟策略更新（Delayed Policy Updates）

Actor网络更新频率低于Critic网络（例如每2步更新一次Actor），避免策略在Critic未收敛时产生错误引导。

#### 3. 目标策略平滑（Target Policy Smoothing）

在目标动作上添加小的随机噪声，防止Actor过度拟合Q函数的尖峰：

$$
a' = \mu_{\theta'}(s') + \text{clip}(\epsilon, -c, c)
$$

### 4.2 TD3算法伪代码

```
初始化：两个Actor网络 μθ1, μθ2，两个Critic网络 Qω1, Qω2
目标网络：μ'theta ← μθ1, Q'ω1 ← Qω1, Q'ω2 ← Qω2

for episode in range(M):
    初始化噪声 N，环境
    for t in range(T):
        a = μθ1(s) + N_t
        执行获取 r, s'

        存储到回放缓冲区

        if t % d == 0:  # 每d步更新一次
            采样batch
            a' = μ'theta(s') + clip(ε, -c, c)
            y = r + γ * min(Q'ω1(s',a'), Q'ω2(s',a'))
            更新两个Critic网络

            if t % (2*d) == 0:  # Actor更新更慢
                更新Actor：∇θJ = ∇aQω1(s,μθ(s))∇θμθ(s)
                软更新目标网络
```

## 5. 实现细节与经验

### 5.1 关键实现要点

1. **动作范围映射**：Actor网络输出通常在[-1, 1]，需要根据环境action_space映射到实际范围：
   ```python
   action = action * (env.action_space.high - env.action_space.low) / 2 + (env.action_space.high + env.action_space.low) / 2
   ```

2. **目标网络软更新**：
   ```python
   for target_param, param in zip(target.parameters(), online.parameters()):
       target_param.data.copy_(tau * param.data + (1 - tau) * target_param.data)
   ```

### 5.2 常见问题与调参经验

根据实际实现经验，DDPG/TD3常见问题：

1. **Q值过估计**：训练过程中Q值可能急剧上升，导致策略崩溃
   - 解决：使用TD3的双Q网络和目标策略平滑

2. **训练不稳定**：rewards先变大后变小
   - 解决：使用较小的学习率，TD3的延迟更新策略

3. **探索不足**：动作噪声衰减过快
   - 解决：保持恒定的探索噪声，或使用递减策略

### 5.3 超参数推荐

| 参数 | DDPG | TD3 |
|------|------|-----|
| Actor学习率 | 1e-4 | 1e-3 |
| Critic学习率 | 1e-3 | 1e-3 |
| 折扣因子γ | 0.99 | 0.99 |
| 目标网络更新τ | 5e-3 | 5e-3 |
| 批量大小 | 256 | 256 |
| 噪声σ | 0.1 | 0.1 |
| 目标噪声裁剪c | - | 0.5 |
| 延迟更新频率d | - | 2 |

## 6. PyTorch代码实现

以下是基于PyTorch的完整实现，包含Actor、Critic、经验回放和训练逻辑。



### 6.2 Actor网络

Actor网络输入状态，输出一个确定性动作（使用tanh将输出限制在[-1, 1]）：

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class Actor(nn.Module):
    def __init__(self, input_size, output_size, net_width=128):
        super(Actor, self).__init__()
        self.layer1 = nn.Linear(input_size, net_width)
        self.layer2 = nn.Linear(net_width, net_width)
        self.output_layer = nn.Linear(net_width, output_size)

    def forward(self, obs):
        actv1 = F.relu(self.layer1(obs))
        actv2 = F.relu(self.layer2(actv1))
        return torch.tanh(self.output_layer(actv2))  # 输出范围 [-1, 1]
```

### 6.3 Q-Critic网络

Critic网络输入状态和动作的组合，输出Q值：

```python
class QCritic(nn.Module):
    def __init__(self, obs_size, action_size, output_size, net_width=128):
        super(QCritic, self).__init__()
        self.layer1 = nn.Linear(obs_size + action_size, net_width)
        self.layer2 = nn.Linear(net_width, net_width)
        self.output_layer = nn.Linear(net_width, output_size)

    def forward(self, obs, action):
        sa = torch.cat([obs, action], 1)  # 状态与动作拼接
        actv1 = F.relu(self.layer1(sa))
        actv2 = F.relu(self.layer2(actv1))
        return self.output_layer(actv2)
```

### 6.4 DDPG实现

```python
class DDPG(nn.Module):
    def __init__(self, env, action_noise=0.1, sampled_mini_batch_size=256,
                 gamma=0.99, rou=0.995, learning_rate=0.005, replay_size=100000):
        super(DDPG, self).__init__()
        self.env = env
        self.action_space_low = env.action_space.low
        self.action_space_high = env.action_space.high
        self.action_noise = action_noise
        self.gamma = gamma
        self.rou = rou  # 软更新系数

        self.state_size = env.observation_space.shape[0]
        self.action_size = env.action_space.shape[0]

        # 创建网络
        self.actor = Actor(self.state_size, self.action_size)
        self.actor_optimizer = optim.Adam(self.actor.parameters(), lr=learning_rate)

        self.critic = QCritic(self.state_size, self.action_size, 1)
        self.critic_optimizer = optim.Adam(self.critic.parameters(), lr=learning_rate)

        # 目标网络
        self.target_actor = Actor(self.state_size, self.action_size)
        self.target_actor.load_state_dict(self.actor.state_dict())
        self.target_critic = QCritic(self.state_size, self.action_size, 1)
        self.target_critic.load_state_dict(self.critic.state_dict())

        # 经验回放
        self.replay_memory = ReplayMemory(replay_size)

    def get_action(self, obs, noise_scale=None):
        """获取动作（含探索噪声）"""
        mean = self.actor(obs)
        mean_action = mean.detach().numpy().flatten()
        noise_scale = noise_scale if noise_scale is not None else self.action_noise
        noise = np.random.normal(0, noise_scale, size=mean_action.shape)
        action_with_noise = mean_action + noise
        action_with_noise = np.clip(action_with_noise, -1, 1)
        # 映射到环境动作空间
        action_with_noise = action_with_noise * (self.action_space_high - self.action_space_low) / 2 \
                            + (self.action_space_high + self.action_space_low) / 2
        return action_with_noise

    def get_target_action(self, obs):
        """获取目标网络输出的动作"""
        mean = self.target_actor(obs)
        action = mean.clamp(-1, 1)
        # 映射到环境动作空间
        action = action * (self.action_space_high_tensor - self.action_space_low_tensor) / 2 \
                 + (self.action_space_high_tensor + self.action_space_low_tensor) / 2
        return action

    def train_model(self):
        """训练一轮"""
        transitions = self.replay_memory.sample(self.sampled_mini_batch_size)
        transitionBatch = Transition(*zip(*transitions))

        # 准备batch数据
        states_batch = torch.cat(transitionBatch.state)
        actions_batch = torch.cat(transitionBatch.action)
        rewards_batch = torch.cat(transitionBatch.reward).unsqueeze(1)
        next_obs_tensor = torch.cat(transitionBatch.next_state)

        # 计算当前Q值
        current_Q_values = self.critic(states_batch, actions_batch)

        # 计算目标Q值
        max_actions = self.get_target_action(next_obs_tensor)
        values_target = self.target_critic(next_obs_tensor, max_actions)
        y = rewards_batch + self.gamma * values_target

        # 更新Critic
        critic_loss = F.mse_loss(current_Q_values, y)
        self.critic_optimizer.zero_grad()
        critic_loss.backward()
        torch.nn.utils.clip_grad_norm_(self.critic.parameters(), 1.0)
        self.critic_optimizer.step()

        # 更新Actor（策略梯度：最大化Q值）
        latest_actions = self.actor(states_batch)
        actor_loss = -(self.critic(states_batch, latest_actions)).mean()
        self.actor_optimizer.zero_grad()
        actor_loss.backward()
        torch.nn.utils.clip_grad_norm_(self.actor.parameters(), 1.0)
        self.actor_optimizer.step()

        # 软更新目标网络
        for target_param, param in zip(self.target_actor.parameters(), self.actor.parameters()):
            target_param.data.copy_(self.rou * target_param.data + (1 - self.rou) * param.data)
        for target_param, param in zip(self.target_critic.parameters(), self.critic.parameters()):
            target_param.data.copy_(self.rou * target_param.data + (1 - self.rou) * param.data)
```

### 6.5 TD3实现（双Q网络+延迟更新+目标策略平滑）

```python
class DoubleQCritic(nn.Module):
    """双Q Critic，用于TD3"""
    def __init__(self, obs_size, action_size, output_size, net_width=128):
        super(DoubleQCritic, self).__init__()
        # 第一个Q网络
        self.critic1_layer1 = nn.Linear(obs_size + action_size, net_width)
        self.critic1_layer2 = nn.Linear(net_width, net_width)
        self.critic1_output_layer = nn.Linear(net_width, output_size)
        # 第二个Q网络
        self.critic2_layer1 = nn.Linear(obs_size + action_size, net_width)
        self.critic2_layer2 = nn.Linear(net_width, net_width)
        self.critic2_output_layer = nn.Linear(net_width, output_size)

    def forward(self, obs, action):
        sa = torch.cat([obs, action], 1)
        # Q1
        c1_actv1 = F.relu(self.critic1_layer1(sa))
        c1_actv2 = F.relu(self.critic1_layer2(c1_actv1))
        q1 = self.critic1_output_layer(c1_actv2)
        # Q2
        c2_actv1 = F.relu(self.critic2_layer1(sa))
        c2_actv2 = F.relu(self.critic2_layer2(c2_actv1))
        q2 = self.critic2_output_layer(c2_actv2)
        return q1, q2


class TD3(nn.Module):
    def __init__(self, env, action_noise=0.1, sampled_mini_batch_size=200,
                 gamma=0.99, rou=0.995, learning_rate=0.005, replay_size=1000000,
                 target_noise_clip=0.5, target_policy_noise=0.2):
        super(TD3, self).__init__()
        self.gamma = gamma
        self.rou = rou
        self.target_noise_clip = target_noise_clip
        self.target_policy_noise = target_policy_noise
        # ... 其他初始化与DDPG类似 ...

        # 双Q Critic
        self.double_critic = DoubleQCritic(self.state_size, self.action_size, 1)
        self.double_critic_optimizer = optim.Adam(self.double_critic.parameters(), lr=learning_rate)

    def get_target_action(self, obs):
        """目标策略平滑：在目标动作上添加噪声"""
        mean = self.target_actor(obs)
        noise = torch.randn_like(mean) * self.target_policy_noise
        noise = noise.clamp(-self.target_noise_clip, self.target_noise_clip)
        action = (mean + noise).clamp(-1, 1)
        # 映射到环境动作空间
        action = action * (self.action_space_high_tensor - self.action_space_low_tensor) / 2 \
                 + (self.action_space_high_tensor + self.action_space_low_tensor) / 2
        return action

    def train_model(self, updateActor=True):
        """TD3训练：Critic每步更新，Actor延迟更新"""
        # ... 数据采样 ...

        # 获取双Q值
        current_Q1, current_Q2 = self.double_critic(states_batch, actions_batch)

        # 目标Q值：取较小值（截断双Q）
        target_Q1, target_Q2 = self.double_critic_target(next_obs_tensor, max_actions)
        y = rewards_batch + self.gamma * torch.min(target_Q1, target_Q2)

        # 更新Critic（两个Q网络都要更新）
        critic_loss = F.mse_loss(current_Q1, y) + F.mse_loss(current_Q2, y)
        self.double_critic_optimizer.zero_grad()
        critic_loss.backward()
        self.double_critic_optimizer.step()

        # 延迟更新Actor
        if updateActor:
            latest_actions = self.actor(states_batch)
            actor_loss = -(self.double_critic(states_batch, latest_actions)[0]).mean()
            self.actor_optimizer.zero_grad()
            actor_loss.backward()
            self.actor_optimizer.step()
```

## 8. 总结

DDPG是将深度学习与确定性策略梯度结合的里程碑式算法，为连续动作空间控制提供了有效解决方案。TD3通过三项关键改进显著提高了训练稳定性和性能，是目前处理连续控制问题的默认选择之一。
